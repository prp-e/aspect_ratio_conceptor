# Aspect Ratio Conceptor 

This is just a toolbox written to handle the aspect ratio in image generation models with no control over the size of images.

## How to use the repository 

This repo is not meant to be used as an standalone, individual piece of code. However, if you need to use it, just do this:

- Set up a python virtual environment: `python3 -m venv .v` 
- Activate your environment: `source .v/bin/activate`
- Install dependencies: `pip install -r requirements.txt`
- Use it with Python's interactive interface: `python -i main.py`
